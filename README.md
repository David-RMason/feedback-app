# Feedback App

A feedback application built with React to showcase modern state management, routing, and animation techniques.

## Quick Start

```bash
# Install dependencies
npm install

# Run app + mock API server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Tech Stack

- **React 19** - Functional components with hooks
- **React Context API** - Global state management
- **React Router v7** - Client-side routing
- **Framer Motion** - Smooth component animations
- **json-server** - Mock REST API

## Features

- Full CRUD operations (Create, Read, Update, Delete)
- REST API integration with async/await
- Global state management without prop drilling
- Multi-page navigation with React Router
- Real-time form validation
- Smooth animations on feedback items

## Project Structure

```
src/
├── components/     - Reusable React components
├── context/        - FeedbackContext for global state
├── routes/         - Home & About pages
└── data/           - Mock database (db.json)
```

## Learn More

This project demonstrates:
- Component architecture with React Context
- Async data fetching and API integration
- Form handling with validation
- Animation implementation with Framer Motion
- Client-side routing patterns
